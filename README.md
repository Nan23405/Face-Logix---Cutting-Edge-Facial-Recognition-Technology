# Face Recognition System

A Python-based face recognition application using Tkinter GUI and PIL for image processing.

## Requirements
- Python 3.7+
- tkinter
- Pillow (PIL)

## Installation

```bash
pip install pillow
```

## Project Structure
```
Face Recognition Way 2/
├── First.py              # Main application file
├── college_images/       # Training images directory
└── README.md             # This file
```

## Usage

Run the application:
```bash
python First.py
```

## Features
- GUI-based interface using Tkinter
- Image processing capabilities
- Face recognition functionality
